JupyterLab extension providing syntax highlighting for Terraform files. Open `.tf` and `.tfvars` files directly in JupyterLab with proper syntax recognition.

## Features

- **Terraform file type registration** - Recognizes `.tf` and `.tfvars` file extensions
- **Syntax highlighting** - HCL syntax highlighting for Terraform configuration files
- **File browser integration** - Terraform files displayed with appropriate icons

## Requirements

- JupyterLab >= 4.0.0

## Install

```bash
pip install jupyterlab_terraform_file_type_extension
```

## Uninstall

```bash
pip uninstall jupyterlab_terraform_file_type_extension
```
